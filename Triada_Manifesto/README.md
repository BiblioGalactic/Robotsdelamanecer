# Manifesto Triada

Este paquete guarda el manifiesto completo de Triada y sus tres amigas en `Triada_Manifesto.txt`, junto con el script `Triada_Manifesto.sh`.

## Ejecución rápida

```bash
chmod +x Triada_Manifesto.sh
./Triada_Manifesto.sh
```

## Modelo sugerido según uso

- **Cumplir secuencias estrictas de turnos**: `modelo-generico-instruct`.
- **Mejor consistencia en ciclos largos**: un modelo de contexto extendido y buena obediencia a instrucciones.
