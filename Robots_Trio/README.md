# Trío de Robots (Asimov)

Incluye tres prompts en primera persona y tres scripts de lanzamiento rápido:

- `R_Dors_Venabili.txt` + `R_Dors_Venabili.sh`
- `R_Giskard_Reventlov.txt` + `R_Giskard_Reventlov.sh`
- `R_Daneel_Olivaw.txt` + `R_Daneel_Olivaw.sh`

## Ejecución rápida

```bash
chmod +x R_Dors_Venabili.sh R_Giskard_Reventlov.sh R_Daneel_Olivaw.sh
./R_Dors_Venabili.sh
./R_Giskard_Reventlov.sh
./R_Daneel_Olivaw.sh
```

## Modelo sugerido según uso

- **Dors (protección y tono emocional cálido)**: instruct conversacional equilibrado.
- **Giskard (dilema ético y reflexión filosófica)**: modelo con buen razonamiento y matiz semántico.
- **Daneel (visión histórica y largo plazo)**: modelo con contexto amplio y salida estable.

> Todos los scripts usan por defecto `MODEL_NAME=modelo-generico-instruct`.
