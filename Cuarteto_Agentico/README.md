# Cuarteto Agentico

Incluye el prompt completo en `Cuarteto_Agentico.txt` y un script de lanzamiento rápido en `Cuarteto_Agentico.sh`.

## Ejecución rápida

```bash
chmod +x Cuarteto_Agentico.sh
./Cuarteto_Agentico.sh
```

## Modelo sugerido según uso

- **Razonamiento general y coordinación multiagente**: `modelo-generico-instruct` (default del script).
- **Mayor profundidad y seguimiento de reglas largas**: usa un modelo instruct de contexto amplio (32k+ tokens).
