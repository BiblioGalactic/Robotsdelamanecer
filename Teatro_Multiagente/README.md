# Teatro Multiagente

Contiene una plantilla reutilizable para diálogos multiagente en `Teatro_Multiagente_Base.txt` y el lanzador `Teatro_Multiagente.sh`.

## Ejecución rápida

```bash
chmod +x Teatro_Multiagente.sh
./Teatro_Multiagente.sh
```

## Modelo sugerido según uso

- **Prototipos y pruebas rápidas**: `modelo-generico-instruct`.
- **Estilo narrativo y creatividad alta**: un modelo orientado a escritura creativa.
